# Русский dashboard

Цель: сделать интерфейс понятным на русском языке.

Тексты для действий:

- BUY_WATCH -> Присмотреться
- SELL_WATCH -> Осторожно
- HOLD_WATCH -> Ждать
- AVOID_WATCH -> Пропустить

Смысл блоков:

- Что произошло с монетой за 24 часа.
- Какая оценка модели.
- Какой риск.
- Какие новости рядом.
- Почему система дала такую идею.

Важно: интерфейс должен показывать идеи для анализа, а не торговые команды.
